# Aster · AI Office 智能助手

> 让 PowerPoint、Excel、Word 在你手里像聊天一样高效。

## 一句话定位

把 DeepSeek + AI 生图直接装进原生 Office —— 不用切网页、不用复制粘贴、不用为昂贵的 Copilot 订阅买单。

---

## 核心能力 · 九大场景覆盖三大宿主

### PowerPoint 篇 · 从想法到成稿，只需 3 步

- **一键生成 PPT 大纲**
  把月报文字稿、会议笔记或产品介绍丢进 Aster，瞬间生成 N 页幻灯片大纲，标题与要点一气呵成。原来 2 小时的活儿，30 秒搞定。

- **AI 智能配图**
  选中任意 slide，Aster 自动理解内容并给出配图——可选 AI 实时生图，也可从开源图库一键检索。再也不用切到搜索引擎找图、再判断版权。

- **大段文字一键瘦身**
  把堆砌的长段落自动重写为简洁的要点结构，让 PPT 看起来专业、易读。汇报现场不再被吐槽"幻灯片太满"。

### Excel 篇 · 不再为公式与数据加班

- **自然语言转公式**
  说出你想要的——"算每个门店近 30 天均值"，Aster 给你可直接粘贴的标准 Excel 公式，含相对/绝对引用。新手秒变 Excel 老炮。

- **公式翻译官 + 报错调试医生**
  看不懂别人写的复杂公式？粘进来 Aster 逐段解释。报错 #REF! / #VALUE! 不知所措？Aster 告诉你错在哪、应该怎么改。

- **数据清洗一句话搞定**
  "拆分地址为省/市/区"、"统一日期格式"、"去掉首尾空格"——选中数据说出需求，Aster 自动清洗并预览，确认后写回。

### Word 篇 · 写作助理，永不拖稿

- **多风格一键润色**
  同一段文字一键切换"严谨 / 口语化 / 简洁 / 抒情 / 抒文"等多种风格，找到最合适的表达。对外邮件、汇报材料、合同条款都能稳稳拿下。

- **长文一键 TL;DR**
  长报告、合同、论文太长不想啃？Aster 帮你提炼 3 句话精华 + 关键要点列表。10 分钟读完别人 1 小时的稿子。

- **大纲秒变长文**
  给一个标题、几条要点，Aster 生成结构完整的多段落文档草稿，省下数小时码字时间。从此提案、周报、报告不再拖到 deadline。

## 跨宿主通用能力

- **全格式文件理解**：Word / Excel / PPT / PDF / TXT / Markdown / CSV / 图片——丢给 Aster，让它基于文件内容生成回答或新内容。截屏发表也能"看图说话"。
- **多轮对话感知上下文**：像 ChatGPT 一样在 Office 内部聊天，Aster 实时感知你的选中文本、当前 slide、当前单元格位置。
- **流式输出**：AI 边想边写，逐字呈现，告别"漫长加载"的焦虑。
- **一键写回文档**：AI 输出可直接插入到当前 slide / 单元格 / 段落，不再复制粘贴丢格式。

## 快速上手

1. 在 Edge 或 Chrome 中安装 Aster Add-in（开源、免费）。
2. 在 DeepSeek 与 aihubmix 控制台获取 API Key（5 分钟内完成）。
3. 把 Key 粘贴到 Aster 设置面板。
4. 打开任意 PPT / Excel / Word，让 AI 开始替你干活。
