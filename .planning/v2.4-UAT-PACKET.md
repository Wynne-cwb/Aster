## 🎯 北极星场景（最高优先 — Phase 26 配置可移植）

这是 v2.4 的核心价值验证，**务必先验**：

- [ ] **跨 partition/跨宿主导入还原零重输**
  1. 在宿主 A（如 PowerPoint）Settings → 新分区「配置备份与迁移」→「导出配置」→ 下载 `aster-config-YYYYMMDD.json`
  2. 打开文件确认含：Provider 配置 + **明文 API key** + 默认 Provider + 选区附件开关 + 用户偏好 + 主题强调色 + Pexels key + 生图默认模型；**不含**聊天历史 / 引导已读 / Pexels Worker baseURL
  3. 在**另一浏览器 / 另一宿主（如 Word 或 Excel，不同 partition）/ 干净环境**（红条提示「未配置 Key」）→「导入配置」→ 选该文件 → 简单确认对话框（含明文警告重申）→ 完成 toast 摘要
  4. **期望：全部配置还原，红条 banner 消失，无需重输任何 key**

- [ ] **常驻警告四要素**：分区常显警告，文案含「明文 API 密钥 / 妥善保管 / 用完即删 / 勿通过不安全渠道传输」四要素齐全（常驻醒目即可，非强制勾选）
- [ ] **同 id Provider 覆盖确认**：导入含同 id Provider 时 → 单独确认对话框（取消 / 跳过冲突项 / 覆盖并导入 三选）；选「跳过」时被跳 Provider 的 key **不被覆盖**
- [ ] **损坏/非法 JSON**：导入非 JSON / 损坏 / 非 Aster 配置文件 → 可操作错误对话框（message+hint，不崩溃）
