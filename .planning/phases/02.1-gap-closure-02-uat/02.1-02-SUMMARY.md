## 对齐数学校核

改动前：
- 胶囊文本左边界 = inputbar(16) + pill-row(16) + pill 自身 padding(2) = 34px
- 输入框文本左边界 = inputbar(16) + composer 边框(1) + composer-left(12) = 29px
- **差 5px，肉眼可见**

改动后：
- 胶囊文本左边界 = inputbar(16) + pill-row(8) + pill 自身 padding(2) = 26px
- 输入框文本左边界 = inputbar(16) + composer 边框(1) + composer-padding(8) = 25px
- **差 1px，在视觉可见阈值（3px）以内**
