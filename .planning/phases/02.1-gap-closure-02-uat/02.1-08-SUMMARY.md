## What Was Built

**G-08 D-31/D-32/D-33/D-34 完整实现：**

1. **眼睛 toggle（D-31）**：SelectionPill 新增眼睛按钮（EyeIcon/EyeOffIcon），`attachEnabled=true` 显 EyeIcon，`false` 显 EyeOffIcon + 胶囊整体 `opacity: 0.55`（is-disabled）。胶囊在两种状态下始终渲染（不再 return null）。

2. **持久化（D-32）**：`providerStore.setAttachEnabled` 写入 `STORAGE_KEYS.SELECTION_ATTACH_ENABLED = 'aster:selection:attachEnabled'`。`hydrateFromStorage` 优先读新 key，旧 key `SELECTION_AUTO_ATTACH` 存在则一次性迁移写入新 key。D-32 字面约束 `selection_attach_enabled`（snake_case）在 `storage.ts` 注释显式标注，说明仓库 namespace 前缀为统一约定，D-32 语义全部兑现。

3. **× 临时隐藏（D-33）**：× 按钮调用 `onDismiss` 回调，InputBar 中 `selectionDismissed` 状态 session-only，不持久化；发送后 reset。

4. **handleSend guard（D-34）**：InputBar.handleSend 通过 `useProviderStore.getState().attachEnabled` 读取状态，`false` 时 `sel = undefined`，不调用 `adapter.getSelection()`（T-02.1-08-01 缓解）。

5. **双向绑定**：SettingsPanel「自动附带选区」开关 → `setAttachEnabled`；SelectionPill 眼睛 → `setAttachEnabled`；同一 providerStore state，双向联动。
