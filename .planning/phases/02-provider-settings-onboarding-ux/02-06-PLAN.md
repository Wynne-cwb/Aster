<threat_model>
## Trust Boundaries

| Boundary | Description |
|----------|-------------|
| AI 输出 → react-markdown 渲染 | AI 内容是不可信字符串，通过 react-markdown 渲染（自动转义，不用 dangerouslySetInnerHTML） |
| error.message → UI 展示 | AsterError.message 不直接渲染到 UI（用 ERROR_UI_MAP 的 reason 替换） |
| adapter.insert → Office.js | 插入内容经 Office.js API 写回，不经 Aster 服务器 |

## STRIDE Threat Register

| Threat ID | Category | Component | Disposition | Mitigation Plan |
|-----------|----------|-----------|-------------|-----------------|
| T-02-21 | Tampering | AI 输出通过 react-markdown 渲染，可能含 HTML | mitigate | react-markdown 默认禁用原始 HTML（rehype-raw 未引入）；不用 dangerouslySetInnerHTML；代码块不注入 script |
| T-02-22 | Information Disclosure | ErrorBubble 直接展示 error.message（可能含 Key） | mitigate | ErrorBubble 只展示 ERROR_UI_MAP[errorCode].reason（固定字符串），不展示 message prop 内容；T-01-04 覆盖 |
| T-02-23 | Spoofing | CostBadge 徽章金额可被 AI 响应内容影响 | accept | costCny 由 calcCostCny(usage, providerId) 计算，usage 来自 Provider 的 SSE usage chunk，不来自 AI 生成文本 |
| T-02-24 | Information Disclosure | SelectionPill 展示选区元数据（slide 序号/单元格地址/字数）| accept | 这些信息本就是用户当前文档状态，展示在 Task Pane 内属于正常功能；不发送到任何服务器 |
