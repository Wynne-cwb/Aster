**类型定义区（顶部 export）：**

```typescript
export const ASTER_CONFIG_VERSION = 1;

export interface AsterConfigExport {
  app: 'aster';
  version: number;
  exportedAt: string;   // ISO 8601
  data: AsterConfigData;
}

export interface AsterConfigData {
  providers: ProviderConfig[];
  keys: Record<string, string>;   // key = provider.id；'pexels' 为 Pexels key 的固定 id
  defaultProviderId: string;
  selectionAttachEnabled: boolean;
  userPreferences: string;
  brandAccentColor: string;
  pexelsKey: string;
  imageGenModel: string;
}

export type ImportErrorCode =
  | 'INVALID_JSON'
  | 'NOT_ASTER_CONFIG'
  | 'UNSUPPORTED_VERSION'
  | 'EMPTY_CONFIG';

export interface ImportError {
  code: ImportErrorCode;
  message: string;
  hint: string;
}

export interface ImportResult {
  providerCount: number;
  keyCount: number;
  prefsRestored: boolean;
}
```

**纯函数层（可直接单测，无副作用）：**

`buildExportData()`: 读 useProviderStore.getState().providers → 遍历每个 provider 用 storage.get(STORAGE_KEYS.KEY_PREFIX + p.id) 收集 keys → 单独读 storage.get(STORAGE_KEYS.PEXELS_API_KEY) 为 pexelsKey → 读其余 storage key → 返回 AsterConfigExport 对象。**仅收集 D-02 锁定字段，绝对不收集 CHAT_HISTORY_PREFIX / ONBOARDING_SEEN / PEXELS_BASE_URL。**

`exportConfig()`: 调 buildExportData() → JSON.stringify(data, null, 2) → new Blob([json], {type:'application/json'}) → URL.createObjectURL → 临时 `<a>` 点击下载 → revokeObjectURL。文件名：`aster-config-${new Date().toISOString().slice(0,10).replace(/-/g,'')}.json`。

`parseImportFile(raw: string)`: 返回 `{ ok: true; config: AsterConfigExport } | { ok: false; error: ImportError }`。JSON.parse 异常 → INVALID_JSON。parsed.app !== 'aster' 或缺 version/data → NOT_ASTER_CONFIG。parsed.version > ASTER_CONFIG_VERSION → UNSUPPORTED_VERSION。data.providers 为空数组且 Object.keys(data.keys).length === 0 → EMPTY_CONFIG。

`detectConflicts(imported: AsterConfigData, currentProviders: ProviderConfig[]): string[]`: 返回 imported.providers 中与 currentProviders 存在同 id 的 provider id 列表。

   ```typescript
   function upsertProviderById(config: ProviderConfig): void {
     const store = useProviderStore.getState();
     const exists = store.providers.find((p) => p.id === config.id);
     const updated = exists
       ? store.providers.map((p) => (p.id === config.id ? { ...config } : p))
       : [...store.providers, config];
     useProviderStore.setState({ providers: updated });
     storage.set(STORAGE_KEYS.PROVIDERS, updated);
   }
   ```
   对 config.providers 中不在 skipIds 的每个 provider 调 upsertProviderById()。
