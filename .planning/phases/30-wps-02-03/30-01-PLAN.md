## Trust Boundaries

| Boundary | Description |
|----------|-------------|
| 用户输入 → probe.js fetch | Key 从 `<input type="password">` 读取后仅用于 Authorization header，不写 localStorage，不打印到日志 |
| probe.js → DeepSeek/aihubmix/Pexels | 纯浏览器直连，无 Aster 自有服务器中转（Core Value 硬约束：无后台） |

## STRIDE Threat Register

| Threat ID | Category | Component | Disposition | Mitigation Plan |
|-----------|----------|-----------|-------------|-----------------|
| T-30-01 | Information Disclosure | API Key → clipboard 报告 | mitigate | `generateReport()` 不输出原始 Key；Key 仅作为 Authorization header 值，report 只记录 `pass/fail + rawValues（不含 key 字段）` |
| T-30-02 | Information Disclosure | API Key → localStorage | mitigate | Key 输入框值不调用 `localStorage.setItem`；`checkLocalStorageWrite()` 仅写哨兵值（`aster:probe:sentinel`），不写 Key |
| T-30-03 | Spoofing | 探针冒充 Aster 后台 | accept | 探针纯浏览器直连，无后台可冒充；Core Value 保证（声明在 index.html 注释） |
| T-30-04 | Tampering | 恶意 jsplugins.xml 替换 | accept | GitHub Pages 托管，受 GitHub 账户保护；仓库 push 需用户认证 |
| T-30-05 | Information Disclosure | Key 明文可见 | mitigate | 三个 Key 输入框均 `type="password"`，防屏幕录制/截图意外泄露 |
