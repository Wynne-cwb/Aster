## Trust Boundaries

| Boundary | Description |
|----------|-------------|
| Office 宿主 → manifest 声明的资源 URL | manifest 内所有 URL（Task Pane / commands / icon）指向 GitHub Pages 公开 HTTPS |
| 用户 sideload manifest → Office 运行时 | 用户手动加载 manifest，Office 据此拉取 Task Pane HTML |

## STRIDE Threat Register

| Threat ID | Category | Component | Disposition | Mitigation Plan |
|-----------|----------|-----------|-------------|-----------------|
| T-01-09 | Tampering | manifest 资源 URL | mitigate | 所有 manifest URL 强制 HTTPS（wynne-cwb.github.io，GitHub Pages 默认 HTTPS）；不引入 http:// 或第三方非微软域名作为脚本来源。AppDomains 仅列 deepseek/aihubmix（Phase 2 直连 Provider 用，符合无后台直连约束——不经任何 Aster 自有服务器）。 |
| T-01-10 | Spoofing | sideload manifest GUID | mitigate | 用正式唯一 GUID 替换 spike 占位 GUID，避免与其它 add-in 的 ID 冲突导致加载混乱。 |
| T-01-11 | Information Disclosure | commands.ts / ShowTaskpane | accept | Phase 1 ribbon 仅打开 Task Pane，不传递任何参数/凭证；无 Key、无外发。Phase 4-6 function command 上线时再评估传参面。 |
