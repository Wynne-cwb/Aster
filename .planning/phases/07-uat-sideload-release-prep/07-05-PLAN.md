## SC1: PPT Topic→Deck（PowerPoint）

**准备：** 打开 PowerPoint，新建空白演示文稿（或只有标题页）
**输入 prompt（复制粘贴）：**
> 帮我做一份「Q3 销售复盘」PPT，给 leadership 看，重点华东

**期望行为：**
1. Task Pane 显示 AgentControlBar（暂停 + 中止 + 步骤计数）
2. Aster 依次调用 read tools → insert_slide → set_shape_text，8-15 步完成
3. 每步 AgentControlBar 显示差异化文案（不是统一「思考中...」）
4. 完成后 DiffLogPanel 展示 N 张卡片（中文 humanLabel）
5. 可以点某张卡片的「撤销该步」或「撤销本次所有操作」

**验收点：**
- [ ] 实际在 PPT 里创建了 ≥3 张 slides
- [ ] DiffLogPanel 卡片文案是中文（不是 raw tool name）
- [ ] 步数 ≤20（不超 max_steps 软着陆）
- [ ] NFR-03：首 token ≤2s（从 Send 到 Task Pane 出现第一个字），单步 ≤10s

**证据记录：**
- 步数：____
- 端到端耗时：____
- DiffLogPanel 截图：____（路径或「已截图」）

**期望行为：**
1. Aster get_used_range_summary → set_range_values（可选清洗）→ apply_formula / insert_chart → 三句话总结
2. 完成后 DiffLogPanel 有 Excel 操作卡片

**验收点：**
- [ ] Excel 里出现了图表 或 formula 结果
- [ ] Task Pane 给出了三句话洞察
- [ ] DiffLogPanel 展示操作记录

## SC4: PPT Shape 精细化（PowerPoint）

**准备：** 打开 PowerPoint，某张 slide 左下角有一张图片（可插入任意图片后拖到左下角）
**输入 prompt：**

**期望行为：**
1. Aster list_slides → get_slide → list_shapes_on_slide → 根据 (left, top) 推断「左下角」

**验收点：**
- [ ] 图片边框变为红色（或 Aster 完成对应操作）
- [ ] 图片位置右移（left 坐标变化 ≥5 点）
- [ ] 步数 ≤6（精细操作，不应 runaway）
