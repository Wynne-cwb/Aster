    **组件内部状态：**

    **「确认插入」按钮 onClick**（D-02 解耦：调 insertImage helper）：
    ```typescript
    const handleInsert = async () => {
      setStatus('inserting');
      // adapter 从 agentStore/adapterRef 取（executor 确认获取方式）
      const result = await insertImage(host, adapter, base64, mimeType, {
        slideIndex, runId, humanLabel,
      });
      if (result.ok) {
        onInserted();
      } else {
        setError(result.error?.message ?? '插入失败，请重试');
        setStatus('preview');
      }
    };
    ```

    **「重新生成」按钮 onClick**（D-05：同 prompt 重 roll）：
    1. 设 `status='regenerating'`
    2. 重新调 ProviderRegistry.resolve('image-gen')（或 storage.get PREF_IMAGE_GEN_MODEL）
    3. 覆盖 config.model = selectedModelId（预览卡内联切 model，D-04）
    4. 调 `new AihubmixImageClient().generate(prompt, config, {})`
    5. 成功：调 `onRegenerate(result.base64, result.mimeType, selectedModelId)` → 父组件替换预览
    6. 失败：setError + setStatus('preview')

    **「取消」按钮 onClick**：直接 `onCancelled()`（父组件清除 base64 内存态）
