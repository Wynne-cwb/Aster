## What Was Built

### Task 1: icons.tsx
- `base.strokeWidth: 1.75 → 1.5`（全局修改，所有图标同步更新）
- 新增 `GearIcon`：齿轮造型（现有 SettingsIcon 是 sliders 造型），InputBar tools 行使用；支持独立 `size` 和 `strokeWidth` props 覆盖
- 新增 `PaperclipIcon`：回形针附件，InputBar 禁用态按钮
- 新增 `ChevronDownIcon`：向下箭头，D-05 折叠卡；CSS `.is-up` 时 rotate(180deg) 表示展开
- 新增 `ChevronLeftIcon`：向左箭头，Settings header 返回按钮
- 新增 `AlertCircleIcon`：警示圆圈，pane-banner 缺 Key 提示
- 新增 `DocumentIcon`：文件文本图标，selpill 前缀

### Task 2: chat.ts + formatTime.ts
- `Message` 接口新增 `ts?: number`（可选，不破坏现有调用）
- `sendMessage` 的 user message push 补上 `ts: Date.now()`
- 创建 `src/utils/formatTime.ts`：`formatTime(ts: number): string` → "MM-DD HH:MM" 格式，供 ChatBubble Wave 3 时间戳渲染
