**1. [Rule 2 — 缺少 size prop] EyeIcon / EyeOffIcon / SendIcon / StopIcon 补加 size prop**
- **发现于：** Task 1 — SelectionPill 重皮时传入 `size={11}` 给 EyeIcon/EyeOffIcon
- **问题：** icons.tsx 中 EyeIcon、EyeOffIcon、SendIcon、StopIcon 无 `size?: number` 参数，TypeScript 会报错

- **文件修改：** src/components/icons.tsx
- **Commit：** 377035d

**2. [Rule 1 — Bug] send-btn disabled 逻辑修复（D-04 保留原有行为）**
- **发现于：** Task 2 — InputBar.test.tsx agent 状态守卫测试失败

- **修复：** 恢复为 `disabled={isAgentBusy || !text.trim()}`
- **文件修改：** src/components/InputBar.tsx
- **Commit：** 5606c86
