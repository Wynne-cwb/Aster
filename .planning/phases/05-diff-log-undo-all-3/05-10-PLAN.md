    **SC2 — DiffLogPanel 汇总卡：**
    - Agent 写 3 处 → run 完成后 Task Pane 末尾出现「本次改动 3 处」折叠卡（non-constant）
    - 展开后每步显示中文 humanLabel（不是 raw JSON / tool name）
    - 折叠/展开交互正常（chevron rotate）

    **SC3 — undo all + 手动改防御：**
    - Agent 写 5 处 → 手动在 Word 文档修改其中 1 处内容 → 点「撤销本次所有操作」
    - 弹二次确认 modal → 点「确认撤销」
    - 弹总结 modal：显示「已回滚 4 步，跳过 1 步（你已手动修改）」（精确数字）
    - 文档核验：4 处 agent 改动消失，1 处手改内容保留

    **SC4 — per-step 单步撤销任意顺序：**
    - Agent 写 3 处 → 先点第 2 步「撤销该步」→ 行变 .is-undone
    - 然后点第 1 步「撤销该步」→ 行变 .is-undone
    - 两步都正确撤回（文档内容与顺序符合预期），非总是删「最后一段」

    **SC5 — Settings 双入口：**
    - 打开 Settings → 找到「复制本次操作记录」按钮 → 点击 → 粘贴确认：含操作记录 + 无 API Key

    **SC6 — copy step log 脱敏：**
    - 主界面 InputBar「复制操作记录」按钮（Clipboard 图标）→ 点击 → 2s 反馈「已复制 ✓」
    - 粘贴到文本编辑器确认：
      - 含 user / assistant / tool 三角色消息
      - 含 humanLabel 描述（如「在文档末尾追加段落…」）
      - 无 API Key（无 `sk-` 前缀字符串）
